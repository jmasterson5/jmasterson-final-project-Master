battle classes level 1

drifter

Drifter offers a balanced variety of Arts, but no stat bonuses. The standard equipped Weapons are Knives and Assault Rifles. Once the Class Rank reaches 10, this Class can be changed to more specialized Striker, Commando, or Enforcer. The basic Arts of the Drifter are Assault Hammer and Flame Grenade. Drifter has five skill slots, though the class has no skills of its own.

battle classes level 2

striker

Striker (known as Assault in the Japanese version) is a class featured in Xenoblade Chronicles X. It is one of three classes that can be changed from Drifter. The Striker is specialised in offensive and defensive Arts types. This class grants HP and melee attack bonuses. The basic equipped Weapons are Longsword and Assault Rifle. Once the Class Rank reaches 10, it gives access to the specialised Samurai Gunner or Shield Trooper. The basic Arts of the Striker are Rising Blade, Defensive Stance and Power Dive. Striker has only one skill slot, the lowest of any class.

commando

Commando (known as Command in the Japanese version) is a class featured in Xenoblade Chronicles X. It is one of three classes that can be changed from Drifter. The Commando increases several Arts damage against enemies. This class provides bonuses to melee accuracy, ranged accuracy, and evasion. The basic equipped Weapons are Dual Swords and Dual Guns. Once the Class Rank reaches 10, Cross gains access to the specialized Winged Viper or Partisan Eagle. The basic Arts of the Commando are Back Slash, Early Bird and Stream Edge.

enforcer

Enforcer (known as Forcer in the Japanese version) is a class featured in Xenoblade Chronicles X. It is one of three classes that can be changed from Drifter. The Enforcer is specialised in Support and Ranged Arts. This class provides bonuses to ranged attack and potential. The basic equipped Weapons are Knives and Rayguns. Once the Class Rank is complete, it gives access to the specialised to Psycorruptor or Blast Fencer. The basic Arts of the Enforcer are Beam Barrage and Repair.

The playable characters of this class include Phog, whereas Murderess's class is an Alternative Winged Viper.

Psycorrupter

Psycorruptor (known as Psy Seeker in the Japanese version) is a class featured in Xenoblade Chronicles X. It is one of two classes that can be leveled up from Enforcer and can be leveled up to Mastermind. Blast fencer

Blast Fencer is a class featured in Xenoblade Chronicles X. This class provides bonuses to melee attack and ranged attack. Duelist

Duelist (known as General Sword in the Japanese version) is a class featured in Xenoblade Chronicles X. This class provides bonuses to HP and melee attack. It is one of six final classes and can only be leveled up from Samurai Gunner. Nagi's class is an Alternative Duelist. Duelist has three skill slots.

Bastion Warrior

Bastion Warrior (known as Heavy Striker in the Japanese version) is a class featured in Xenoblade Chronicles X. It is one of six final classes and can only be leveled up from Shield Trooper

Full Metal Jaguar

Full Metal Jaguar is a class featured in Xenoblade Chronicles X. This class provides bonuses to melee accuracy, ranged accuracy, and evasion. It is one of six final classes and can only be leveled up from Winged Viper. The playable characters of this class include Elma.

Astral Crusader

Astral Crusader (known as Star Crusader in the Japanese version) is a class featured in Xenoblade Chronicles X. This class provides bonuses to ranged attack and accuracy. It uses Sniper Rifle and Javelin as its weapons. It is one of six final classes and can only be leveled up from Partisan Eagle.

Mastermind

Mastermind (known as Elite Planner in the Japanese version) is a class featured in Xenoblade Chronicles X. It is one of six final classes and can only be leveled up from Psycorruptor.

Galactic Knight

Galactic Knight (known as Galaxy Knight in the Japanese version) is a class featured in Xenoblade Chronicles X. It is one of six final classes and can only be leveled up from Blast Fencer. This class provides bonuses to melee attack and ranged attack.
